Module Install Instructions

To install <% module.documentation or module.module.0.name or module.package %>, simply copy and paste either of the commands in to your terminal

cpanm

                    cpanm <%  module.documentation or module.module.0.name or release.main_module or (release.author _ "/" _ release.archive) %>

CPAN shell

                    perl -MCPAN -e shell
                    install <% module.documentation or module.module.0.name or release.main_module or (release.author _ "/" _ release.archive) %>

For more information on module installation please visit the detailed CPAN module installation guide.
